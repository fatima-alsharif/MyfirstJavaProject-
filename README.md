# نظام إدارة المهام (Task Management System)

هذا المشروع عبارة عن تطبيق بسيط بلغة **Java** لإدارة المهام اليومية، يهدف إلى تطبيق مفاهيم البرمجة كائنية التوجه (OOP).

## مميزات المشروع:
* **إضافة مهام جديدة:** تحديد عنوان لكل مهمة.
* **تتبع الحالة:** معرفة ما إذا كانت المهمة مكتملة أم لا.
* **التاريخ والوقت:** تسجيل وقت إنشاء كل مهمة تلقائياً.

## كيفية التشغيل:
1. تأكد من تثبيت **JDK** على جهازك.
2. قم بتشغيل ملف `TaskManager.java`.

## لغات البرمجة المستخدمة:
* Java
